# TransientX

TransientX is a one command line high performance transient search software.

## Dependencies

- boost > 1.56
- PlotX (https://github.com/ypmen/PlotX)
- sofa

## Install

- ./bootstrap
- ./configure --prefix=[install_path]
- make
- make install

## Usage

- transientx -h for help
